** Simulador de Caixa Eletrônico - Java **

Este projeto é um exercício prático de Lógica de Programação desenvolvido durante a Residência em Software do Serratec. O objetivo é simular as operações básicas de um terminal bancário via console.

** Objetivos **

Este repositório contém a resolução de uma trilha de exercicios progressivos, focados no desenvolvimento de habilidades em Java, abrangendo desde a lógica de programação até conceitos avançados de Programação Orientada a Objetos (POO).

** Exercícios implementados **

Simulador de ATN: ** Controle de fluxo com "do-while" e "switch-case", incluindo regras de limites de saques diários.
Sistema bancário (POO): ** Migração de classes e objetos, uso de construtores para bônusde boas-vindase encapsulamento.
E-commerce: ** Modelagem de relacionamentos entre classes (Cliente, Pedido e Itens) com lógica de cálculode frete.
** Tecnologias Ultilizadas **

Java (JDK 17+)
Git para versionamento
GitHub para hospedagem do código
